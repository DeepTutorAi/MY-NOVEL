# 📖 Key Terms & Vocabulary: บ้านคุซาบิบนเขาคุโระมิโซะ

---

## สถานที่

| คำ | คำอธิบาย | หมายเหตุ |
| :--- | :--- | :--- |
| **เขาคุโระมิโซะ** | เขาขนาดกลางในชนบทญี่ปุ่น ปกคลุมด้วยป่าค่อนข้างทึบ มักมีหมอกลงเร็ว | ชื่อมาจาก "คุโระ" (ดำ) + "มิโซะ" (ถั่วหมัก) หรือ "มิโซะ" ที่หมายถึงที่ลุ่ม/บ่อ |
| **บ้านคุซาบิ** | บ้านร้างบนเขาคุโระมิโซะ เดิมเป็นบ้านพักตากอากาศ | "คุซาบิ" (楔) = ลิ่ม, สลัก — เหมือนลิ่มที่ปักอยู่บนเขา |
| **เรียวกัง** | ที่พักของกลุ่มตัวเอกริมเขาคุโระมิโซะ | ไม่จำเป็นต้องมีชื่อเฉพาะ — เรียกว่า "เรียวกัง" หรือ "ที่พัก" ก็พอ |
| **พิพิธภัณฑ์ท้องถิ่น** | พิพิธภัณฑ์เล็ก ๆ ในหมู่บ้านที่ชิองอาสาดูแล | แหล่งข้อมูลข่าวลือของกลุ่ม |
| **ห้องเจ้าสาว** | ห้องใต้ดินที่ชิองใช้เป็นฉากสุดท้าย | มีชุดกิโมโนขาว ของอายาเมะ เทปเสียง |
| **ห้องควบคุม** | ห้องใต้บ้านที่มีจอมอนิเตอร์ เครื่องอัดเสียง สมุดบันทึก | หัวใจของชิอง |

## ตัวละคร

| คำ | คำอธิบาย |
| :--- | :--- |
| **ชิอง มาโดกะ** | ฆาตกร / ผู้ดูแลเรียวกังและพิพิธภัณฑ์ |
| **อายาเมะ ฮิราซากะ** | คนรักของชิองที่เสียชีวิต |
| **อาราอิ เรียวยะ** | protagonist / POV หลัก |
| **นาโอะ อามามิยะ** |เพื่อนเรียวยะ / catalyst |
| **ไคโตะ ซากุราบะ** | แฟนของริสะ / the protector |
| **ริสะ อาซาฮินะ** | POV รอง / แฟนของไคโตะ |
| **มิซากิ คุโรซาวะ** | เพื่อนสนิทรริสะ / the realist |

## วัตถุ / อุปกรณ์สำคัญ

| คำ | คำอธิบาย |
| :--- | :--- |
| **เทปคาสเซ็ต** | สื่อบันทึกเสียงที่ชิองใช้ — บางม้วนเป็นเหยื่อเก่า บางม้วนเป็นอายาเมะ |
| **ผ้าพันคอของริสะ** | ของที่หายไป — ถูกใช้เป็นเหยื่อล่อ |
| **หน้ากากเจ้าสาว** | หน้ากากไม้ซีด ๆ ที่ชิองใช้ — หน้าหญิงสาว |
| **กิโมโนขาว** | ชุดที่ชิองใช้ตอนเป็นฆาตกร |
| **เสียงกระดิ่ง** | สัญญาณของชิอง — ดังจากชุดฆาตกร |
| **สมุดบันทึก** | ในห้องควบคุม — มีลายมือเดียวกับที่ชิองให้กลุ่มดู |
| **แผนผังบ้าน** | ทั้งของจริงและของปลอมที่ชิองปล่อยไว้ |
| **เทป "กลับบ้าน"** | เทปของอายาเมะที่ชิองกำไว้ตอนตาย |

## ศัพท์เฉพาะเรื่อง

| คำ | คำอธิบาย |
| :--- | :--- |
| **เจ้าสาวภูเขา** | ข่าวลือในหมู่บ้าน — ชื่อที่คนเรียก "ผี" หรือฆาตกร |
| **บ้านคุซาบิ** | ชื่อบ้านร้าง — เป็นทางการ |
| **การทดสอบ** | สิ่งที่ชิองเรียกสิ่งที่เขาทำ — ไม่ใช่ "ฆ่า" ในมุมเขา |
| **ทางหนีแคบ** | ทางเดินในบ้านที่บังคับให้เลือก |
| **ห้องเสียง** | ห้องที่มีเทปเล่นซ้ำ |
| **ห้องกุญแจ** | ห้องที่กุญแจผูกกับศพ |
| **ห้องสองประตู** | ห้องที่บังคับให้เลือกทาง |

## Sensory Vocab Bank (สำหรับใช้ใน prose)

### กลิ่น
- ฝน, ดินเปียก, ไม้เก่า, ฝุ่น, ความชื้น, เชื้อรา, น้ำยาฆ่าเชื้อ, กระดาษเก่า, โลหะ

### เสียง
- ไม้เอี๊ยดอ๊าด, เสียงกระดิ่ง, เสียงเคาะ, เสียงเทป, เสียงลม, เสียงฝน, เสียงหัวใจ, เสียงหายใจ

### สัมผัส
- เย็น, ชื้น, เหนียว, หยาบ, เกรอะ, คม, อับ

### แสง
- มืด, สลัว, ไฟฉาย, หลอดไฟกระพริบ, เทียน, เงา
