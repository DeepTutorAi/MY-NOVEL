### 2. นาโอะ อามามิยะ — Catalyst / Comic Relief

| รายการ | รายละเอียด |
| :--- | :--- |
| **อายุ** | 20 — ชาย — ปี 2 |
| **MBTI** | ENTP |

| **จุดแข็ง** | ไหวพริบดี กล้าลอง คิดนอกกรอบ หาทางออกแปลก ๆ ได้ |
| **จุดอ่อน** | ประมาท เห็นความกลัวเป็นเรื่องเล่น ไม่เคารพขอบเขตของคนอื่น |
| **ความสัมพันธ์เด่น** | เพื่อนสนิทเรียวยะ — ต้นเหตุที่ทำให้กลุ่มขึ้นไปบนเขา |

#### Character Arc
เริ่มต้นเป็นคนที่มองทุกอย่างเป็นเกม สนุกกับความตื่นเต้น ไม่คิดว่าผลของการกระทำจะร้ายแรง

การเปลี่ยนแปลงมาถึงเมื่อมีคนบาดเจ็บหรือตายเพราะการตัดสินใจของเขา — ตอนนั้นเขาจะพังเพราะรู้ตัวว่าเปิดประตูให้เรื่องทั้งหมด

#### ถ้ารอด: เป็นคนที่พังที่สุดทางใจ รับรู้ว่าการลองดีของตัวเองมีราคา
#### ถ้าตาย: จุดเปลี่ยนที่ทำให้กลุ่มหยุดทะเลาะแบบเด็ก ๆ และเข้าใจว่าทุกการเลือกมีราคา

#### ประโยคสำคัญ
> "ก็แค่บ้านร้าง ไม่เห็นต้องซีเรียส"
> (หลังทุกอย่างพัง) "...ฉันพาพวกมึงขึ้นมาจริง ๆ"

### 3. ไคโตะ ซากุราบะ — The Protector

| รายการ | รายละเอียด |
| :--- | :--- |
| **อายุ** | 21 — ชาย — ปี 2 (รุ่นพี่) |
| **MBTI** | ISFJ |
| **ส่วนสูง** | 180 ซม. |
| **จุดแข็ง** | ดูแลคนอื่นดี ใจเย็น รับผิดชอบ เป็นห่วงเป็นใย |
| **จุดอ่อน** | ปกป้องจนควบคุม ตัดสินใจแทนริสะ กลัวเสียคนรักจนมองไม่เห็นว่าตัวเองกำลังบีบเธอ |
| **ความสัมพันธ์เด่น** | แฟนของริสะ |

#### Character Arc
เริ่มต้นด้วยความรักที่หวังดีแต่มี passive control — เขาตัดสินใจแทนริสะตลอดเพราะ "เป็นห่วง"

การเปลี่ยนแปลงมาถึงเมื่อเขาต้องเลือกระหว่าง "ปกป้องริสะ" กับ "ปล่อยให้ริสะเลือกเอง"

ตอนจบ: ถ้ารอด เขาต้องเผชิญว่าความรักของเขาเคยเป็นกรง เขากับริสะยังคบกันแต่ไม่เหมือนเดิม หรือหยุดพักเพื่อให้ริสะได้ยืนเอง

#### ประโยคสำคัญ
> "ริสะอยู่ข้างหลังฉัน"
> (ตอนปล่อยมือ) "...ไปเถอะ เลือกเอง"

### 4. ริสะ อาซาฮินะ — POV รอง

| รายการ | รายละเอียด |
| :--- | :--- |
| **อายุ** | 20 — หญิง — ปี 2 |
| **MBTI** | ESFP |
| **ส่วนสูง** | 165 ซม. |
| **ลักษณะ** | ผมยาวสีทอง ตาสีฟ้า — สวยมาก เป็นจุดสังเกต |
| **จุดแข็ง** | อ่านอารมณ์คนเก่ง ทำให้คนอื่นผ่อนคลาย มีความอ่อนโยน |
| **จุดอ่อน** | อ่อนแอที่สุดในกลุ่ม พึ่งคนอื่น กลัวเลือด กลัวความมืด ไม่ชอบตัดสินใจเอง |
| **ความสัมพันธ์เด่น** | แฟนของไคโตะ — เพื่อนสนิทมิซากิ |

#### Character Arc
เริ่มต้นเป็นคนที่อ่อนแอและถูกดูแล — ไคโตะตัดสินใจแทน มิซากิคอยบอก

การเปลี่ยนแปลง: จากคนที่ร้องไห้และรอคนช่วย เป็นคนที่เริ่มตัดสินใจเล็ก ๆ เอง กล้าขัดไคโตะ กล้าบอกมิซากิว่าไม่ต้องสั่งทุกอย่าง

จุดเปลี่ยนใหญ่: ตอนถูกจับในห้องเจ้าสาว เธอไม่ได้มีพลังวิเศษหรือกล้าหาญทันที แต่เธอใช้สิ่งที่เธอมี — การสังเกตเล็กน้อย และคำพูดที่ตรงไปตรงมา — เพื่อกระแทกชิอง

ตอนจบ: ยังกลัว ยังฝันร้าย แต่เริ่มพูดว่า "ฉันเลือกเอง" ไม่ให้ไคโตะตอบแทนทุกครั้ง ตัดผมสั้นลงหรือเลิกใส่คอนแทกต์ฟ้า — ไม่ใช่เพราะอยากลืม แต่เพราะอยากเป็นตัวเองมากกว่า "ภาพที่คนอื่นมอง"

#### POV Notes
- POV ของริสะใช้เฉพาะช่วงที่เธอถูกแยก / ถูกจับ / คุยกับชิอง
- Internal thought ของเธอจะดิบและอ่อนไหว — ไม่ใช่การวิเคราะห์ แต่เป็นความรู้สึกล้วน ๆ
- เธอสังเกตคนผ่านอารมณ์ ไม่ใช่ผ่านข้อมูล

#### ประโยคสำคัญ
> "ฉันเลือกเอง"
> (พูดกับชิอง) "ถ้าเธอเป็นคนดีขนาดนั้น... คุณไม่เหนื่อยเหรอ ที่ต้องใช้ชื่อเธอมาทำเรื่องแบบนี้ซ้ำ ๆ"

### 5. มิซากิ คุโรซาวะ — The Realist

| รายการ | รายละเอียด |
| :--- | :--- |
| **อายุ** | 21 — หญิง — ปี 2 (รุ่นพี่) |
| **MBTI** | ESTJ |
| **ส่วนสูง** | 168 ซม. |
| **จุดแข็ง** | ฉลาด ตัดสินใจเร็ว กล้าชน วางแผนดี |
| **จุดอ่อน** | แข็งเกินไป ชอบสั่ง ไม่ยอมรับว่ากลัว ขัดแย้งกับเรียวยะและนาโอะ |
| **ความสัมพันธ์เด่น** | เพื่อนสนิทของริสะ — ขัดแย้งกับเรียวยะตลอด |

#### Character Arc
เริ่มต้นเป็นคนที่แข็งและเอาตัวรอด — วางแผน ลงมือ ไม่รอ ไม่ถาม

การเปลี่ยนแปลง: เรียนรู้ว่าการแข็งอย่างเดียวไม่พอ — ต้องฟังคนอื่นด้วย

ตอนจบ: ยังแข็งเหมือนเดิม แต่ฟังคนอื่นมากขึ้น เธอกับเรียวยะไม่ได้กลายเป็นสนิททันที แต่มีความเคารพกัน

#### หมายเหตุพิเศษ
มิซากิดุทุกคน แต่กับริสะจะอ่อนลง เธอรู้ว่าริสะไม่ได้อ่อนแอเพราะโง่ แต่เพราะถูกคนอื่นดูแลจนไม่เคยได้ฝึกยืนเอง

#### ประโยคสำคัญ
> "คุยกันตอนรอดก่อนเหอะ"
> "แกคิดว่านี่คือปริศนาให้แกนั่งแกะ? นี่คือของจริง"

### 6. อายาเมะ ฮิราซากะ — Emotional Anchor

| รายการ | รายละเอียด |
| :--- | :--- |
| **อายุขณะเสียชีวิต** | 24 |
| **MBTI** | INFP |
| **สถานะ** | เสียชีวิต (3 ปีก่อนเหตุการณ์หลัก — ตอนนั้นชิองอายุ 24 ด้วย) |
| **ลักษณะนิสัย** | เป็นคนดีมาก อ่อนโยน แต่ไม่อ่อนแอ INFP ที่เชื่อใน "คนเรามากกว่าสิ่งที่เราเห็นภายนอก" — เธอเป็นนักอุดมคติที่มองเห็นศักยภาพของคนอื่นแม้ในเวลาที่คนนั้นมองไม่เห็นตัวเอง เธอเชื่อว่ามนุษย์เปลี่ยนได้ และนั่นคือสิ่งที่ทำให้เธอเป็นคนเดียวที่เคยทำให้ชิองหยุดตัดสินมนุษย์ |
| **ที่มาของประโยคสำคัญ** | "คนเราไม่ได้มีแค่วินาทีที่แย่ที่สุดของตัวเองนะ ชิอง" — นี่คือ INFP ที่มองคนทั้งคน ไม่ใช่แค่ช่วงที่แย่ที่สุดของเขา |
| **บทบาทในเรื่อง** | Ghost / Emotional Anchor — เป็นคนเดียวที่เคยทำให้ชิองหยุดตัดสินมนุษย์ |

#### ความสำคัญ
อายาเมะคือ INFP ที่เชื่อในศักยภาพของคน แม้แต่ในคนที่ผิดหวังในตัวเอง เธอไม่ใช่แค่คนรักที่ตาย แต่เป็น **เสียงที่คอยบอกว่า "คนเราเปลี่ยนได้"** ที่ชิองพยายามฆ่าตลอด 3 ปี

เมื่อชิองถามคำถามสุดท้ายว่า "คนเราเปลี่ยนได้ไหม?" — เขากำลังถามคำถามที่อายาเมะเคยตอบไว้แล้ว แต่เขาไม่ยอมเชื่อ มันเป็นคำถามที่เขากลัวคำตอบ

#### ประโยคสำคัญ
> "คนเราไม่ได้มีแค่วินาทีที่แย่ที่สุดของตัวเองนะ ชิอง"
> (ป้ายเทป) "กลับบ้าน"
