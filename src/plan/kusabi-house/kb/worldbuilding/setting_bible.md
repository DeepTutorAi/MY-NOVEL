# 🌍 Setting Bible: บ้านคุซาบิบนเขาคุโระมิโซะ

---

## ภาพรวมภูมิศาสตร์

### เขาคุโระมิโซะ (黒味噌山)
| รายละเอียด | คำอธิบาย |
| :--- | :--- |
| **ที่ตั้ง** | ชนบทญี่ปุ่น ห่างจากตัวเมืองหลักประมาณ 2-3 ชั่วโมง |
| **ลักษณะ** | เขาขนาดกลาง ปกคลุมด้วยป่าค่อนข้างทึบ ทางเดินขึ้นเขาเป็นเส้นทางดินและหิน |
| **บรรยากาศ** | ชื้น เย็น โดยเฉพาะช่วงบ่ายแก่ ๆ มักมีหมอกลงเร็ว |
| **ตำนานท้องถิ่น** | ชาวบ้านเชื่อว่ามี "เจ้าสาวภูเขา" สิ้งสถิตอยู่บนเขา ใครขึ้นไปหลังตะวันตกจะไม่ได้กลับลงมา — ข่าวลือที่ชิองสร้างและปล่อยต่อ |
| **เวลาเดินทางขึ้น** | จากเชิงเขาถึงบ้านคุซาบิ ประมาณ 45 นาที - 1 ชั่วโมงเดิน |

### เรียวกัง (ที่พักของกลุ่ม)
| รายละเอียด | คำอธิบาย |
| :--- | :--- |
| **ลักษณะ** | เรียวกังขนาดเล็กถึงกลาง ดูเก่าแต่สะอาด มีบ่อน้ำร้อนเล็ก ๆ |
| **บรรยากาศ** | เงียบ มีแขวน้อย โดยเฉพาะช่วงโลว์ซีซั่น เป็นสถานที่ที่ชิองเลือกเพราะความเปลี่ยว |
| **ทีมงาน** | เจ้าของเรียวกังวัยกลางคน ชิองมาช่วยงานพิเศษ (เป็นหน้ากากของเขา) |
| **ทำเล** | ห่างจากทางขึ้นเขาคุโระมิโซะประมาณ 10 นาทีเดิน |
| **จุดสังเกต** | จากเรียวกัง มองเห็นยอดเขาคุโระมิโซะได้ในวันที่ฟ้าเปิด |

### หมู่บ้าน
| รายละเอียด | คำอธิบาย |
| :--- | :--- |
| **ลักษณะ** | หมู่บ้านเล็ก ๆ ประชากรน้อย ร้านค้าไม่กี่ร้าน |
| **วัฒนธรรม** | คนในหมู่บ้านรู้เรื่องบ้านคุซาบิ แต่ไม่พูดถึงมาก — บางคนกลัว บางคนทำเป็นไม่รู้ |
| **พิพิธภัณฑ์ท้องถิ่น** | มีพิพิธภัณฑ์เล็ก ๆ ที่ชิองอาสาดูแล เป็นแหล่งข้อมูลข่าวลือของกลุ่ม |
| **ระบบขนส่ง** | มีรถโดยสารวันละ 2-3 เที่ยว ไม่มีรถไฟใกล้ |

## 🏚️ บ้านคุซาบิ (บ้านร้างบนเขา)

### ประวัติบ้าน
| หัวข้อ | รายละเอียด |
| :--- | :--- |
| **ชื่อเดิม** | คุซาบิ (楔) — แปลว่าลิ่ม หรือสลัก |
| **ประวัติ** | บ้านไม้เก่าแก่สองชั้น เดิมเป็นบ้านพักตากอากาศของครอบครัวหนึ่ง ต่อมาถูกทิ้งร้างนานกว่า 10 ปี |
| **สภาพก่อนชิอง** | ทรุดโทรม ฝุ่นหนา หน้าต่างแตก มีข้าวของบางส่วนของเจ้าของเดิมหลงเหลือ |
| **สภาพหลังชิอง** | ชิองซ่อมแซมบางส่วน ปรับปรุงระบบประตู-หน้าต่าง ติดตั้งอุปกรณ์เพิ่มเติม โดยเฉพาะห้องใต้ดินและระบบเสียง |
| **ข่าวลือในหมู่บ้าน** | มีผีเจ้าสาว / คนขึ้นไปแล้วหาย / ได้ยินเสียงผู้หญิงร้องตอนกลางคืน |

## บรรยากาศและ sensory notes

### ภาพรวมของบ้าน
- **แสง**: มืด — หน้าต่างส่วนใหญ่ตอกแผ่นไม้ มีหลอดไฟไม่กี่จุด บางจุดใช้เทียน
- **กลิ่น**: ฝุ่น ไม้เก่า ความชื้น เชื้อรา ในห้องใต้ดินมีกลิ่นดินและโลหะ
- **เสียง**: เสียงพื้นไม้เอี๊ยดอ๊าด เสียงลม เสียงกระดิ่งจากชุดฆาตกร เสียงเครื่องเล่นเทป
- **อุณหภูมิ**: ข้างนอกเย็น ข้างในอุ่นกว่าเล็กน้อย แต่ชื้น ห้องใต้ดินเย็นและอับ

### สภาพอากาศ
- **วันที่กลุ่มมาถึง**: ฝนปรอย — ทำให้บรรยากาศอึดอัด
- **วันที่ขึ้นเขา**: เมฆครึ้ม เย็น อาจมีฝนระหว่างทาง
- **คืน climax**: ฝนตกหนัก — ทำให้เสียงเครื่องเล่นเทปกลบและซ่อนเสียงจริง
- **เช้าวันรุ่งขึ้น**: ฝนหยุด อากาศปลอดโปร่ง — ตรงข้ามกับบรรยากาศเมื่อคืน

## 🧠 ปรัชญาบ้าน: "บ้านจำความผิด"

บ้านคุซาบิไม่ใช่เครื่องจักรฆ่า มันคือ **พื้นที่อัดแน่นไปด้วยความทรงจำของความผิดที่คนเคยทำต่อกัน**

### ความแตกต่างจาก Jigsaw/Saw
| Saw | บ้านคุซาบิ |
| :--- | :--- |
| เครื่องจักรทรมาน ออกแบบให้เจ็บ | สภาพแวดล้อมบ้านเก่า + กับดักจิตวิทยา |
| ทดสอบความอดทนทางกาย | ทดสอบความสัมพันธ์และการเลือก |
| เหยื่อต้องทำร้ายตัวเองเพื่อออก | เหยื่อต้อง *เลือก* — ทิ้ง หรือ กลับไป |
| กับดักมีกลไกซับซ้อน | กับดักใช้ของในบ้าน: ประตู, เชือก, เทป, ฝาไม้ |
| จุดจบ = ตายหรือรอด | จุดจบ = รู้ว่าตัวเองเป็นคนแบบไหน |

### ระบบการทำงานของบ้าน
1. **บันทึก** — ทุกเสียง เทป ภาพ ถูกเก็บไว้ เหยื่อเก่ายัง "อยู่" ในบ้านผ่านร่องรอย
2. **ซ้ำ** — เสียงในอดีตถูกเล่นซ้ำตอนคนใหม่มา — ให้คนใหม่ได้ยินว่าคนเก่าตายยังไง
3. **สะท้อน** — กับดักถูกออกแบบให้เหยื่อเห็นตัวเองในคนที่ตายไปแล้ว
4. **ลงโทษตามความผิด** — ห้องที่คนตาย = ห้องที่ความผิดนั้นถูกทำให้เป็นรูปธรรม

### ตัวอย่างการแปลงความผิดเป็นกับดัก
| ความผิดที่เหยื่อทำ | สิ่งที่บ้าน "จำ" | กับดักที่เกิดขึ้น |
| :--- | :--- | :--- |
| ทิ้งคนอื่น | ห้องที่มีรอยลากไปถึงประตู | ต้องเลือกว่าจะกลับไปช่วยหรือเดินหนี |
| ปิดประตูใส่คน | เสียงเคาะประตูที่เล่นซ้ำ | ต้องฟังเสียงนั้นจนกว่าจะแก้รหัส |
| เลือกตัวเองก่อน | กุญแจผูกกับศพ | ต้องแตะศพเพื่อเอากุญแจ |
| โกหกเพื่อรอด | เทปเสียงคำโกหกของตัวเอง | ต้องฟังคำโกหกของตัวเองวนซ้ำ |
