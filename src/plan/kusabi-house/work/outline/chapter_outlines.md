## 📝 Detailed Chapter Outlines: ARC 1

---

### บทที่ 01: เรียวกังฮิโนะโมริ
- **POV**: เรียวยะ
- **Arc**: ARC 1
- **Setting**: เรียวกังกลางชนบท — เย็นวันศุกร์ ฝนปรอย
- **Starting Status**: กลุ่มเพิ่งเดินทางมาถึง ยังรู้สึกว่ามันเป็นทริปพักร้อนธรรมดา
- **Ending Status (The Shift)**: ทุกคนได้ยินข่าวลือเรื่อง "บ้านเจ้าสาวภูเขา" แต่ละคนเก็บไว้ในใจต่างกัน — รอยร้าวในกลุ่มเริ่มปรากฏ
- **Tension Level**: Low

#### Scene 1.1: รถตู้ถึงเรียวกัง
- **Characters Present**: เรียวยะ, นาโอะ, ไคโตะ, ริสะ, มิซากิ
- **Goal / Incentive**: ลงจากรถ เอาของเข้าที่พัก
- **Conflict / Obstacle**: ฝนตกปรอย นาโอะถ่ายคลิป ริสะตื่นเต้น ไคโตะดูแลริสะมากเกิน มิซากิจัดการทุกอย่างเหมือนหัวหน้าทริป เรียวยะเงียบ ๆ คอยสังเกต
- **Outcome / Reaction**: ปูพื้นฐานไดนามิกกลุ่มและรอยร้าว → เรียวยะไม่ชอบความประมาทของนาโอะ, มิซากิไม่ไว้ใจนาโอะ, มิซากิกับเรียวยะขัดกันเรื่องวิธีคิด, ริสะพยายามยิ้มให้ทุกคนไม่ทะเลาะกัน
- **Key Details to Seed**: กลิ่นฝนแบบเฉพาะ, เสียงกระดิ่งลอยมาจากไหนไม่รู้

#### Scene 1.2: อาหารค่ำและชิอง
- **Characters Present**: ทั้ง 5 คน + ชิอง (มาร่วมโต๊ะ)
- **Goal / Incentive**: กินข้าวเย็น ผ่อนคลายหลังเดินทาง
- **Conflict / Obstacle**: ชิองปรากฏครั้งแรกในฐานะคนช่วยงานเรียวกัง/คนดูแลพิพิธภัณฑ์ท้องถิ่น เขาดูนิ่ง ลึก พูดน้อย และสังเกตคนเก่ง
- **Outcome / Reaction**: ชิองพูดถึง "บ้านคุซาบิ" บนเขาคุโระมิโซะ นาโอะสนใจทันที เรียวยะไม่คิดมาก มิซากิขัดหู — แต่ริสะสังเกตว่าชิองพูดถึงบ้านเหมือนพูดถึงคนที่ยังมีชีวิต
- **Key Details to Seed**: ชิองรู้ทางขึ้นเขาดีเกินไป, เสียงกระดิ่งตอนเดินออกจากห้อง

#### Scene 1.3: ห้องพัก
- **Characters Present**: เรียวยะ, นาโอะ (พักห้องเดียวกัน)
- **Goal / Incentive**: เตรียมตัวนอน
- **Conflict / Obstacle**: นาโอะชวนคุยเรื่องบ้านร้าง เรียวยะบอกให้อย่าเพิ่งคิดมาก — ทะเลาะกันเบา ๆ
- **Outcome / Reaction**: นาโอะบอกจะไปสืบเพิ่มพรุ่งนี้ เรียวยะถอนหายใจ — ปูความขัดแย้ง
- **Key Details to Seed**: เรียวยะนอนไม่หลับ ได้ยินเสียงคล้ายผู้หญิงร้องไกล ๆ

### บทที่ 02: ข่าวลือเจ้าสาวบนเขา
- **POV**: เรียวยะ
- **Arc**: ARC 1
- **Setting**: เรียวกังและหมู่บ้าน — เช้าวันเสาร์
- **Starting Status**: กลุ่มตื่นมา เริ่มสำรวจรอบเรียวกัง
- **Ending Status (The Shift)**: แต่ละคนได้ยินข่าวลือบ้านร้างในเวอร์ชันของตัวเอง — ริสะเริ่มรู้สึกเหมือนถูกจับตามอง
- **Tension Level**: Low-Mid

#### Scene 2.1: ข่าวลือหลายเวอร์ชัน
- **Characters Present**: ทั้ง 5 คน, เจ้าของเรียวกัง, ชาวบ้าน
- **Goal / Incentive**: สอบถามข้อมูลบ้านคุซาบิ
- **Conflict / Obstacle**: แต่ละคนได้ยินข่าวลือไม่เหมือนกัน: ถ้าขึ้นไปกลางคืนจะเห็นเจ้าสาว, ถ้าเธอโค้งให้ห้ามโค้งกลับ, มีคนเคยหาย, เคยใช้ทำพิธีบูชายัญ, บางคนบอกว่าเป็นแค่เรื่องหลอก
- **Outcome / Reaction**: นาโอะอยากไป ริสะกลัวแต่แอบสนใจ มิซากิสงสัยว่าชาวบ้านเล่าไม่ตรงกัน เรียวยะจับได้ว่าชิองรู้มากกว่าที่พูด
- **Key Details to Seed**: เจ้าของเรียวกังบอก "อย่าขึ้นไปเลยครับ" — เสียงไม่มั่น

#### Scene 2.2: เงาชุดขาว
- **Characters Present**: ริสะ, ไคโตะ, มิซากิ
- **Goal / Incentive**: กลับมาที่เรียวกังตอนเย็น
- **Conflict / Obstacle**: ริสะเห็นเงาชุดขาวใกล้แนวป่า — หรือคิดว่าเห็น
- **Outcome / Reaction**: เธอไม่กล้าบอกใคร — แต่เริ่มรู้สึกว่ามีคนจับตามอง

### บทที่ 03: ของที่หาย
- **POV**: เรียวยะ
- **Arc**: ARC 1
- **Setting**: เรียวกัง — เช้าวันเสาร์สายถึงบ่าย
- **Starting Status**: ของริสะหายไปจริง — ถูกพบใกล้ทางขึ้นเขาคุโระมิโซะ
- **Ending Status (The Shift)**: กลุ่มตัดสินใจขึ้นไปบ้านคุซาบิ — ไม่ใช่เพราะโง่ แต่เพราะเหตุผลเฉพาะของแต่ละคนกดดัน
- **Tension Level**: Mid

#### Scene 3.1: หายจริง
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: หาของริสะให้เจอ
- **Conflict / Obstacle**: ตามหาทั่วเรียวกังไม่เจอ — นาโอะเจอมันใกล้ทางเดินขึ้นเขา
- **Outcome / Reaction**: กลุ่มเริ่มตระหนักว่ามีคนเอาของไปวางไว้

#### Scene 3.2: แรงผลักของแต่ละคน
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: ตัดสินใจว่าจะทำยังไง
- **Conflict / Obstacle**:
  - นาโอะ: อยากพิสูจน์และถ่ายคลิป
  - ไคโตะ: อยากเอาของริสะกลับมา
  - มิซากิ: ไม่ไว้ใจว่ามีคนจงใจยุ่งกับริสะ
  - เรียวยะ: เห็นว่าจังหวะของหายผิดธรรมชาติ
  - ริสะ: รู้สึกผิดที่ทำให้ทุกคนวุ่นวาย
- **Outcome / Reaction**: ตกลงขึ้นไปตอนบ่าย

#### Scene 3.3: ทางขึ้น
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: ขึ้นไปถึงบ้านคุซาบิ
- **Conflict / Obstacle**: ทางขึ้นลื่นจากฝน บรรยากาศป่าอึดอัด
- **Outcome / Reaction**: ถึงบ้านคุซาบิ — ประตูหน้าเปิดอยู่
- **Key Details to Seed**: บ้านเงียบมาก เชือกเก่าแขวนอยู่ตรงคาน ป้ายเตือนถูกขูดจนอ่านไม่ออก

### บทที่ 04: บ้านคุซาบิ
- **POV**: เรียวยะ
- **Arc**: ARC 1 (จบ Arc)
- **Setting**: บ้านคุซาบิ — ชั้น 1

- **Ending Status (The Shift)**: ประตูหน้าปิดจากด้านนอก — มีคนล็อก
- **Tension Level**: Mid-High

- **Goal / Incentive**: สำรวจบ้าน
- **Conflict / Obstacle**: บ้านมืด ฝุ่นเยอะ พบของแปลก: เทปเสียงเก่า, รูปถ่าย, ชุดกิโมโนขาว, หน้ากากเจ้าสาว, รอยเลือดเก่า, รอยเล็บบนผนัง
- **Outcome / Reaction**: เริ่มรู้ว่าบ้านนี้ไม่ธรรมดา

- **Conflict / Obstacle**: เสียงกระดิ่งดังทั้งที่ไม่มีลม — รอยเท้าใหม่บนพื้นโคลน

- **Key Details to Seed**: รอยเท้าตรงกับรองเท้าของชิอง

#### Scene 4.3: ประตูปิด
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: ออกจากบ้าน
- **Conflict / Obstacle**: ประตูหน้าล็อกจากด้านนอก — เปิดไม่ได้
- **Outcome / Reaction**: ตระหนักว่าติดอยู่ในบ้าน — จบ Arc 1
- **Key Details to Seed**: เรียวยะเริ่มเชื่อมโยงว่านี่ไม่ใช่บ้านผี

## 📝 Detailed Chapter Outlines: ARC 2

### บทที่ 05: ห้องแรก
- **POV**: เรียวยะ
- **Arc**: ARC 2
- **Setting**: บ้านคุซาบิ — ห้องเสียงและโถงกลาง
- **Starting Status**: ติดอยู่ในบ้าน ต้องหาทางออก
- **Ending Status (The Shift)**: เจอกับดักแรก — รู้ว่าใครบางคนกำลังควบคุมบ้านนี้อยู่
- **Tension Level**: High

#### Scene 5.1: ห้องเสียง
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: หาทางออก — มีคนถูกขังในห้องเล็ก
- **Conflict / Obstacle**: ลำโพงเปิดเสียงร้องขอความช่วยเหลือซ้อนกับเสียงจริงของเพื่อน — ต้องแยกว่าเสียงไหนจริง เสียงไหนเทป
- **Outcome / Reaction**: เรียวยะใช้เหตุผลจับจังหวะเสียงเทป นาโอะเริ่มตื่นจริง มิซากิหงุดหงิดที่เรียวยะคิดนานเกินไป ไคโตะสนใจแค่ว่าริสะปลอดภัยไหม ริสะเริ่มรู้ว่าตัวเองเป็นภาระ
- **Key Details to Seed**: กับดักถูกออกแบบมาให้บีบความเชื่อใจ — บ้านจำได้ว่าใครเลือกทางไหน ไม่ใช่ฆ่าทันที

#### Scene 5.2: ไขรหัส
- **Characters Present**: เรียวยะ, มิซากิ
- **Goal / Incentive**: หาวิธีเปิดประตู
- **Conflict / Obstacle**: ประตูล็อกด้วยรหัส เบาะแสอยู่ในเทปที่เล่นซ้ำ
- **Outcome / Reaction**: เรียวยะและมิซากิต้องทำงานร่วมกัน — ทั้งคู่ขัดกันแต่จำเป็นต้องร่วมมือ

### บทที่ 06: ประตูสองทาง
- **POV**: เรียวยะ
- **Arc**: ARC 2
- **Setting**: ทางแยกในบ้านคุซาบิ
- **Starting Status**: กลุ่มเริ่มแตกแยกจากความกดดัน
- **Ending Status (The Shift)**: กับดักบีบให้เลือก — กลุ่มแตกหนัก
- **Tension Level**: High

#### Scene 6.1: ประตูสองทาง
- **Characters Present**: ทั้ง 5 คน (แยกย้าย)
- **Goal / Incentive**: เลือกทาง
- **Conflict / Obstacle**: ทางหนึ่งดูปลอดภัยกว่า อีกทางมีเสียงคนขอความช่วยเหลือ ถ้าเลือกทางปลอดภัย = ทิ้งคนที่ติดอยู่ ถ้าเลือกช่วย = เสียเวลาและเสี่ยงโดนล็อกเพิ่ม
- **Outcome / Reaction**: ไคโตะพยายามพาริสะไปทางปลอดภัย มิซากิโกรธ นาโอะเถียงว่าไม่ควรกลับไปเสี่ยง เรียวยะลังเลเพราะข้อมูลไม่พอ
- **Key Details to Seed**: ชิองไม่ได้ต้องการฆ่าทันที — เขาต้องการเห็น "ใครถูกเรียกชื่อก่อน"

#### Scene 6.2: ช่วยติดอยู่ — แต่ไว้ใจลดลง
- **Characters Present**: ทั้ง 5 คน
- **Goal / Incentive**: ช่วยคนที่ติดอยู่
- **Conflict / Obstacle**: ทุกคนร่วมมือกันช่วยคนออกมาได้ แต่ความไว้ใจกันลดลง
- **Outcome / Reaction**: รอยร้าวลึกขึ้น — ไคโตะเริ่มปิดข้อมูลบางอย่าง ริสะเริ่มรู้สึกว่าตัวเองทำให้ทุกคนเลือกยาก

### บทที่ 07: ศพที่บอกคำตอบ
- **POV**: เรียวยะ
- **Arc**: ARC 2
- **Setting**: ห้องเก็บของและห้องกุญแจ — บ้านคุซาบิ
- **Starting Status**: กำลังหาทางออกอื่น
- **Ending Status (The Shift)**: พบร่องรอยเหยื่อเก่า — รู้ว่านี่คือ crime scene ที่ถูกใช้ซ้ำ
- **Tension Level**: Very High

#### Scene 7.1: ร่องรอยเหยื่อ — หลักฐานของความผิด
- **Characters Present**: เรียวยะ, มิซากิ, นาโอะ
- **Goal / Incentive**: หาของใช้หรือข้อมูล
- **Conflict / Obstacle**: ในห้องเก็บของมีร่องรอย: รอยลากยาวจากกลางห้องถึงประตูที่เปิดไม่ได้ = ใครบางคนถูกลากกลับเข้าไป, มือหุ่นไม้ถูกมัดกับกุญแจเก่า = เหยื่อเคยเลือกทิ้งคนอื่นเพื่อเอาของ, เทปเสียงติดป้ายชื่อคนที่ไม่มีในทะเบียน, ผนังด้านในมีรอยขีดนับวัน, ช่องใต้พื้นที่เล็กพอให้คนเข้าไปติด
- **Outcome / Reaction**: เรียวยะนับร่องรอย — บ้านหลังนี้จำได้ทุกคนที่ตายที่นี่

#### Scene 7.2: ห้องกุญแจ

### บทที่ 09: หลังริสะหาย — Recovery Sequence
- **POV**: เรียวยะ
- **Arc**: ARC 3
- **Setting**: บ้านคุซาบิ — ห้องนอนรวมชั้น 2 ถึงห้องควบคุม
- **Starting Status**: ริสะหายไป กลุ่มแตก — ทุกคนกำลังจะพัง
- **Ending Status (The Shift)**: เรียวยะใช้ประสบการณ์เก่า — เข้าใจคนที่ "พัง" — ดึงมิซากิกลับมา → ช่วยนาโอะด้วยกัน → ดึงไคโตะกลับมาจากความเห็นแก่ตัว → **นี่คือเหตุการณ์ที่ทำให้ชิองตัดสินใจใช้ Final Gambit**
- **Tension Level**: Very High → สะสมสู่ Peak

#### Scene 9.1: เรียวยะ — รู้จักคนที่ "พัง" เพราะเคยถูกมองข้ามมาก่อน
- **Characters Present**: เรียวยะ (เดี่ยว — ครุ่นคิด)
- **Goal / Incentive**: รวบรวมสติ
- **Conflict / Obstacle**: เครียดหนัก — คิดว่าถ้าตัวเองต่อข้อมูลได้เร็วกว่านี้ ริสะคงไม่หาย — เริ่มหลุด: อ่านบันทึกข้ามบรรทัด, มือสั่นตอนจับแว่น, ยอมรับครั้งแรกว่า "ผมไม่รู้"
- **Outcome / Reaction — Pivot Point**: เรียวยะหยุด — หายใจ — แล้วคิดถึงตอน ม.ปลาย ตอนนั้นก็ไม่มีใครฟังเขาเหมือนกัน แต่เขามีเพื่อนหนึ่งคนที่แค่...ฟัง เขาเข้าใจแล้วว่าตอนนี้สิ่งที่กลุ่มต้องการไม่ใช่ "คนที่มีคำตอบ" — แต่เป็น "คนที่อยู่ด้วยตอนที่ไม่มีใครมีคำตอบ"
- **Key Details to Seed**: นี่คือเรื่องจริงในใจเรียวยะ — ไม่ใช่ Monologue ยาว แต่เป็นภาพแวบเดียว: ห้องเรียนว่างตอนเย็น, เพื่อนที่นั่งด้วยโดยไม่พูดอะไร, ความเงียบที่ช่วยเขาได้ตอนนั้น

#### Scene 9.2: ดึงมิซากิ — "ฉันก็ไม่รู้ — แต่เราไปช่วยนาโอะก่อนนะ"
- **Characters Present**: เรียวยะ, มิซากิ
- **Goal / Incentive**: เรียวยะ: ดึงมิซากิ / มิซากิ: กำลังพัง — นั่งพึมพำ
- **Conflict / Obstacle**: มิซากิพังชัดที่สุด: "ริสะกลัวที่แคบ... ริสะไม่ชอบเสียงดัง... ถ้าเธอร้องไห้ เธอจะหายใจไม่ทัน... ฉันบอกแล้วว่าอย่าแยก... ฉันบอกแล้ว..." — เธอใช้การพูดซ้ำเป็นกลไกควบคุมตัวเอง
- **Outcome / Reaction**: เรียวยะนั่งลงข้างเธอ — ไม่ปลอบ ไม่บอกว่าเดี๋ยวก็ดีขึ้น — แต่พูดสิ่งที่เขาเรียนรู้จากอดีต:
  > "ฉันก็ไม่รู้เหมือนกัน... ว่าเราจะช่วยเธอได้ไหม — แต่ตอนนี้นาโอะเจ็บอยู่ ถ้าเราปล่อยเขาไว้ เขาอาจตายก่อนที่เราจะช่วยริสะด้วยซ้ำ — เราไปหาเขาก่อนนะ"
  
  มิซากิหยุดพึมพำ — มองเรียวยะ — สิ่งที่ทำให้เธอกลับมาได้ไม่ใช่ "ความหวัง" แต่คือ **"งานที่ทำได้ตอนนี้"**
  เธอพยักหน้า — ไม่พูด — แต่ลุกขึ้น

#### Scene 9.3: ช่วยนาโอะ — "นายยังไม่ได้ฆ่าใคร — แค่พาพวกเรามา"
- **Characters Present**: เรียวยะ, มิซากิ, นาโอะ
- **Goal / Incentive**: พยุงนาโอะ — ตรวจดูอาการ
- **Conflict / Obstacle**: นาโอะบาดเจ็บ + จมกับความผิด: "ถ้าฉันไม่พูดเรื่องบ้านนั่นตั้งแต่แรก... กูพาพวกมึงมาจริง ๆ"
- **Outcome / Reaction**: เรียวยะกับมิซากิช่วยกันทำแผล — มิซากิไม่ด่านาโอะแล้ว เธอแค่พูดสั้น ๆ: "อย่าขยับ" — เรียวยะพูดกับนาโอะ:
  > "นายยังไม่ได้ฆ่าใครนะ — แค่พาพวกเรามา — ยังแก้ได้"
  
  นี่คือสิ่งที่เรียวยะเรียนรู้ — **คนที่ผิดไม่ต้องการการให้อภัย แต่ต้องการรู้ว่ายังแก้ไขได้**
  นาโอะเงียบ — น้ำตาคลอ — แต่พยักหน้า

- **Shion's Observation**: 🎭 *ชิองกำไมค์แน่น — นี่คือครั้งแรกใน 3 ปีที่มีคนพูดกับ "คนที่ผิด" แบบนี้ — ทุกครั้งเหยื่อจะสาปแช่งคนที่พาพวกเขามา — แต่นี่...เรียวยะไม่ได้ด่า — เขาพูดว่า "ยังแก้ได้" — ชิองถอดหูฟัง — นั่งเงียบ — ครั้งแรกที่เขาเริ่ม "กลัว" ผลลัพธ์*

#### Scene 9.4: ดึงไคโตะ — "ริสะต้องการคนที่เชื่อว่าเธอทำเองได้ ไม่ใช่คนที่ทำให้แทน"
- **Characters Present**: เรียวยะ, มิซากิ, ไคโตะ
- **Goal / Incentive**: ดึงไคโตะกลับมาจากความเห็นแก่ตัว
- **Conflict / Obstacle**: ไคโตะยังกักข้อมูล — กล่าวหาเรียวยะว่าคิดช้า — กล่าวหามิซากิว่าปกป้องริสะไม่ได้ — เขาพร้อมจะพุ่งไปช่วยริสะคนเดียว
- **Outcome / Reaction**: มิซากิจะระเบิดใส่ แต่เรียวยะยกมือห้าม — แล้วพูด:
  > "ถ้าแกไปคนเดียว... แล้วตาย — ริสะจะรู้ว่าแกตายเพราะแกไม่เชื่อใครเลย — นั่นคือสิ่งที่แกอยากให้ริสะจดจำเหรอ"

  ไคโตะชะงัก — เรียวยะพูดต่อ:
  > "ริสะกำลังพยายามยืนด้วยตัวเอง — เธอไม่ต้องการคนที่ทำให้แทน — เธอต้องการคนที่เชื่อว่าเธอทำเองได้ — แกจะเชื่อเธอมั้ย?"

  นี่คือประโยคที่แทงใจไคโตะที่สุด — เพราะเขารู้ว่าที่ผ่านมาเขาไม่เคยเชื่อว่าริสะทำเองได้เลย
  ไคโตะทรุด — กำมือแน่น — แล้วพูด: "ฉัน...จะเชื่อเธอ"

- **Shion's Observation — Cracks Deepen**: 🎭 *ชิองมองจอ — ตอนนี้ทั้งสี่คนกลับมารวมตัวกัน ไม่ได้แตก — แต่แข็งแกร่งกว่าเดิม ไม่ใช่เพราะพวกเขาไม่กลัว — แต่เพราะพวกเขากลัวแล้วยังเลือกที่จะสู้ต่อ นี่คือ moment ที่ชิองรู้ว่า "การทดสอบตามปกติ" จะใช้ไม่ได้อีกแล้ว เขาลุกขึ้น — เปิดลิ้นชัก — หยิบแผน "สุดท้าย" ออกมา — Final Gambit*

### บทที่ 10: เสียงของริสะ

- **Ending Status (The Shift)**: เธอถามชิอง "พวกเราทำอะไรผิด" — คำถามนี้กระทบเขา
- **Tension Level**: Peak

#### Scene 10.1: ห้องเจ้าสาว
- **Characters Present**: ริสะ, ชิอง (พูดผ่านลำโพง)
- **Goal / Incentive**: ริสะพยายามทำความเข้าใจ
- **Conflict / Obstacle**: ชิองยังไม่ให้เห็นหน้าชัด พูดเล่นกับเหยื่อ "เธอไม่ได้ผิดอะไรเป็นพิเศษหรอก แค่เหมาะกับคำถามที่สุด"
- **Outcome / Reaction**: ริสะกลัว แต่ถามทั้งที่เสียงสั่น: "พวกเราทำอะไรผิดไป ถึงต้องตายกันล่ะ"

#### Scene 10.2: คำถามที่กระทบ — ชิองข่มขู่ที่นี่คือการป้องกันตัวเอง
- **Characters Present**: ริสะ, ชิอง
- **Goal / Incentive**: รอคำตอบ
- **Conflict / Obstacle**: ประโยคของริสะกระแทกชิองเพราะมันใกล้กับสิ่งที่อายาเมะเคยเตือนเขา — เสียงลำโพงเงียบ, เขาเผลอกำเทปแน่นขึ้น, ไม่ตอบทันที, ลมหายใจเปลี่ยนไป
- **Outcome / Reaction — ชิองตอบด้วยความรุนแรง เพราะกลัวว่าเธอ "ถูก"**:
  
  ตอนนี้ชิองไม่ได้แค่โกรธ — แต่เขากำลัง **ป้องกันตัวเอง** จากการถูกสั่นคลอน นี่คือกลไกป้องกันตัว — ถ้าเขายอมรับว่าริสะไม่ผิด = เขาต้องยอมรับว่าเขาฆ่าคนบริสุทธิ์มา 3 ปี

  เขาตอบกลับด้วยน้ำเสียงที่ "รุนแรงเกินเหตุ" — เพื่อข่มให้ริสะเงียบ และเพื่อบอกตัวเองว่าเธอผิด:
  
  > "ผิดสิ — พวกเธอแค่ยังไม่รู้ว่าตัวเองจะผิดตอนไหน — เพราะยังไม่มีใครถูกเลือกให้ตาย — ยังไม่มีใครต้องตัดสินใจว่าระหว่าง 'เพื่อน' กับ 'ตัวเอง' จะเลือกอะไร — แล้วเธอจะเห็นเอง...ว่าเพื่อนของเธอจะเรียกชื่อใครก่อนกัน"
  
  **สิ่งที่ชิองไม่รู้ตัว**: การที่เขาต้อง "ข่ม" เธอด้วยคำพูดยาวขนาดนี้ — คือการสารภาพว่าเขาไม่แน่ใจแล้ว — สำหรับคนที่เคยตอบแค่ "ใช่" หรือ "ไม่ใช่" นี่คือสัญญาณว่าเขากำลังสั่นคลอน
  
  หลังจากพูด เขารีบเดินออกจากห้อง — ไม่ใช่เพื่อไปทำแผนต่อ — แต่เพื่อหนีจากเสียงของเธอ

- **Shion's Observation**: 🎭 *ชิองปิดประตูห้องเจ้าสาว — หลังพิงผนังนอกห้อง — หายใจแรง — เขาไม่ได้กลัวริสะ — เขากลัวว่าเธอ "ถูก" — และถ้าเธอถูก — 17 ศพที่ผ่านมา...คืออะไร? — นี่คือครั้งแรกที่เขาไม่กล้ามองจอ*

#### Scene 11.1: เชื่อมโยงเบาะแส — เรียวยะเปิดโปง

- **Outcome / Reaction**: เรียวยะ: "ชิอง... ชิองคือคนที่อยู่กับเรามาตลอด"

#### Scene 11.2: Shion's Final Gambit — "ทางออกมีอยู่ — แต่..."
- **Characters Present**: เรียวยะ, มิซากิ, นาโอะ, ไคโตะ
- **Goal / Incentive**: หาทางช่วยริสะ
- **Conflict / Obstacle**: ทันใดนั้นเสียงชิองดังขึ้นผ่านลำโพง — น้ำเสียงเขาพยายามสงบ แต่มีความกดดันแฝง:
  > "ผมเปิดประตูหน้าให้แล้ว — ทางลงเขาปลอดภัย — กลับไปเถอะครับ"
  
  กลุ่มเงียบ — ทุกคนรู้ว่านี่ไม่ใช่ "ความปรานี"
  
  ชิองพูดต่อ — เสียงเขาสั่นนิด ๆ — เหมือนกำลังพนันทุกอย่าง:
  > "แต่...มีเงื่อนไข — คนที่ต้องอยู่ในบ้าน เพื่อให้คนอื่นรอด — คือหนึ่งในพวกคุณ — ไม่ใช่ริสะ — ผมสัญญาว่าจะปล่อยริสะ ถ้ามีใครสักคน...ยอมอยู่แทนเธอ"
  
  **นี่คือ Final Gambit ของชิอง**: หลังจากเห็นว่ากับดักปกติใช้ไม่ได้ผล — กลุ่มแข็งแกร่งเกินไป — เขาต้องการ "คำตอบ" แบบสุดท้าย เขาเสนอทางรอด **โดยให้พวกเขาเลือกทิ้งใครสักคน** — ไม่ใช่เพราะริสะอ่อนแอที่สุดอีกต่อไป — แต่เพราะ **ถ้าใครสักคนยอมอยู่ — ชิองจะได้พิสูจน์ว่ายังไงก็ต้องมีคนถูกทิ้ง**
  
  เขาหวังว่า:
  - ไคโตะจะเลือกอยู่ — เพราะรักริสะ
  - หรือมิซากิจะเลือกอยู่ — เพราะเป็นเพื่อนสนิท
  - หรือทุกคนจะทะเลาะกันจนแตก — เพราะไม่มีใครอยากอยู่
  - **ยังไงก็ตาม — "หนึ่งคนต้องถูกทิ้ง" — และนั่นจะทำให้ Final Gambit ของเขาสำเร็จ**
  
  **แต่สิ่งที่เกิดขึ้นคือ**:
  
  - **นาโอะ**พูดก่อน — เสียงสั่นแต่จริงจัง: "ฉันอยู่ — มันเป็นความผิดฉันตั้งแต่แรก"
  - **ไคโตะ**สวนทันที — ครั้งแรกที่เขาไม่คิดถึงแค่ริสะ: "ไม่ — แกอยู่ — ฉันจะไปเอาริสะ แล้วทุกคนกลับ"
  - **มิซากิ** — เงียบ — แล้วพูด: "ไม่มีใครอยู่ — เราไปเอาริสะ แล้วออกไปด้วยกัน — ไม่งั้นก็อยู่ด้วยกัน"
  - **เรียวยะ** — มองทุกคน — แล้วพูด: "เราไม่เล่นเกมของเขา — เราเลือกทางของเราเอง"
- **Outcome / Reaction**: ไม่มีใครถูกทิ้ง — ทุกคนเลือก "กลับไปด้วยกัน" — และนี่คือจุดที่แนวคิดของชิองพังอย่างสมบูรณ์
- **Key Details to Seed**: ชิองไม่ตอบ — เงียบไปนาน — เสียงลมหายใจผ่านลำโพง — เหมือนคนที่เพิ่งเห็นสิ่งที่ตัวเองเชื่อมาทั้งชีวิต...ถูกทำลายต่อหน้าต่อตา

### บทที่ 12: บ้านปิดประตูใส่เจ้าของ — Final Gambit ล้มเหลว
- **POV**: เรียวยะ
- **Arc**: ARC 3 (จบ Arc — Climax)
- **Setting**: ห้องเจ้าสาวและทางหนี — บ้านคุซาบิ
- **Starting Status**: Final Gambit ล้มเหลว — ไม่มีใครเลือกทิ้งใคร — ชิองโกรธและสิ้นหวัง ต้องเผชิญหน้ากลุ่มโดยตรง
- **Ending Status (The Shift)**: ริสะทำสิ่งเล็ก ๆ ที่ชิองดูถูก — กับดักทำงานผิด — ชิองบาดเจ็บ — บ้านพัง — กลุ่มหนี
- **Tension Level**: Climax

#### Scene 12.1: ห้องเจ้าสาวแตก — ชิองหมดความอดทน
- **Characters Present**: ทั้งกลุ่ม + ชิอง
- **Goal / Incentive**: ช่วยริสะ
- **Conflict / Obstacle**: ชิองไม่ได้รออย่างสงบอีกแล้ว — เขาโกรธ — Final Gambit ล้มเหลวและนั่นคือสิ่งที่เขากลัวที่สุด — ริสะทำสิ่งเล็ก ๆ ที่ชิองไม่คาด: แก้เชือกได้บางส่วน, ทำให้กลไกขัดจังหวะ, ถ่วงเวลาชิองด้วยคำพูด, ใช้ของของอายาเมะทำให้ชิองเสียจังหวะ
- **Outcome / Reaction**: ชิองเสียสมาธิ — กลุ่มบุกเข้ามา — ปะทะกัน — **ตอนนี้ชิองไม่ใช่ผู้ทดสอบอีกต่อไป — แต่เป็นคนที่กำลังแพ้และรู้ตัว**

#### Scene 12.2: ชิองบาดเจ็บ — Final Gambit กลายเป็นศพของแผนสุดท้าย
- **Characters Present**: ทั้งกลุ่ม + ชิอง
- **Goal / Incentive**: หนี
- **Conflict / Obstacle**: ชิองพยายามฆ่า — แต่ระบบบ้านเริ่มพัง — กับดักเด้งกลับมาใส่เขา — เขาพยายามบังคับทุกอย่างให้กลับมาเข้าทาง — แต่มือสั่น — โกรธ — ควบคุมระบบได้แย่ลง
- **Outcome / Reaction**: ชิองเสียเลือด — แผนสุดท้ายของเขาคือ "ถ้าไม่ได้คำตอบที่ต้องการ ก็ไม่มีใครรอด" — แต่มันสายเกินไป — บ้านทรยศเขาเหมือนที่เขาทรยศความจริงมาตลอด 3 ปี

#### Scene 12.3: คำถามสุดท้าย — "คนเราเปลี่ยนได้ไหม?"
- **Characters Present**: เรียวยะ, ชิอง
- **Goal / Incentive**: เรียวยะ — พาริสะหนี / ชิอง — หยุดพวกเขา
- **Conflict / Obstacle**: ชิองเสียเลือดมาก ขยับแทบไม่ไหว แต่ยังมีแรงถาม — เสียงไม่ใช่ผู้ทดสอบอีกต่อไป มันคือเสียงของคนที่อยากรู้จริง:
  > "ทำไม... ทำไมพวกแกถึงกลับไปช่วยกันได้"

  คำถามนี้ไม่มีความโกรธในตอนแรก — มีแต่ความไม่เข้าใจบริสุทธิ์ เหมือนเด็กที่เพิ่งรู้ว่ากฎที่เขาเชื่อมาตลอดอาจไม่จริง

  เรียวยะตอบ — ไม่ใช่คำตอบสวยหรู ไม่ใช่ปรัชญา: "เพราะเรายังเลือกที่จะกลับมาได้"

  ชิองนิ่งไป แล้วใบหน้าเขาเปลี่ยน — จากคนที่ไม่เข้าใจ เป็นคนที่เริ่มเข้าใจ และความเข้าใจนั้นเผาเขา

  **เขาระเบิด**

  ไม่ใช่การระเบิดแบบอาละวาด แต่เป็นการระเบิดของคนที่กลั้นสามปี:
  - "สามปี... สามปีที่ผมรอ... รอว่าเมื่อไหร่จะมีสักกลุ่มที่เลือกต่างออกไป"
  - "ทุกคนที่เข้ามา... ทุกคน... เลือกทิ้งกัน เลือกปิดประตู เลือกโกหกเพื่อรอด เลือกตัวเองก่อนเสมอ"
  - "ผมให้นรกเขาแค่ทางเลือกเดียวกับที่อายาเมะมี... และทุกคนก็เลือกแบบเดียวกับที่พวกเราทำกับเธอ"
  - เสียงของเขาสั่น — ไม่ใช่ความโกรธเกรี้ยวของฆาตกร แต่เป็นความโกรธของคนที่ผิดหวังในมนุษย์:
  - "บอกผมที... ถ้าคนเราดีจริง... ทำไมสามปีไม่มีใครเลือกรักคนอื่นเท่าตัวเองเลย? ผมผิดหรือที่โลกมันเป็นแบบนี้... หรือโลกผิดที่มันสอนให้รู้ว่าคนเราทิ้งกันได้เสมอ"
  - เสียงของเขาเริ่มแตก — "ผมให้ทางออกพวกแกแล้ว... แค่คนเดียว แค่ยอมอยู่แทนเธอหนึ่งคน... ทำไมพวกแกถึงไม่เอา... ทำไม..."
  - นี่คือการสารภาพว่า Final Gambit คือตั๋วสุดท้ายของเขา — และมันก็ถูกฉีกทิ้งต่อหน้าต่อตา

  จากนั้นเขาหยุด — การขยับทำให้เสียเลือดมากขึ้น เขาหมดแรง เสียงเริ่มแผ่ว

- **Outcome / Reaction**: ชิองทรุดลงกับพื้น หลังพิงผนังห้องเจ้าสาวที่กำลังพัง ในมือยังกองเทป "กลับบ้าน" อยู่ เขาเงียบไปนาน แล้วถามเบา ๆ — เสียงนี้ไม่ใช่ฆาตกร ไม่ใช่ผู้ทดสอบ ไม่ใช่คนวิปริต — แต่มันคือ **ชิองเมื่อ 3 ปีก่อน** คนที่อายาเมะเคยรัก:
  > "คนเรา...เปลี่ยนได้ไหม?"

  ไม่มีใครตอบ

  เรียวยะมองเขา — และชิองไม่รอคำตอบ เพราะเขารู้แล้ว เขาเหลือบมองเทปในมือ รอยยิ้มบาง ๆ ที่ไม่มีความสุข — เป็นรอยยิ้มของคนที่เพิ่งรู้คำตอบแต่สายเกินไปที่จะใช้มัน

  เขาไม่พูดอะไรอีก

  เรียวยะเลือกที่จะพาริสะหนี — ทิ้งชิองไว้กับบ้านที่เขาสร้าง กับเทปที่เขาไม่มีโอกาสฟัง และกับคำถามที่เขาเพิ่งรู้ว่าตัวเองก็เปลี่ยนไม่ได้เหมือนกัน

- **Key Details to Seed**: ชิองไม่ตอบคำถามของตัวเอง — แต่การมองเทปและการยิ้มบาง ๆ บอกทุกอย่าง เขารู้ว่าคนเราเปลี่ยนได้ — อายาเมะพูดถูก — แต่เขาเปลี่ยนไม่ทัน

### Extra E03: คนที่กลับมาทำลายหลักฐาน
- **POV**: ชิอง มาโดกะ
- **Timeline**: 3 ปีก่อน — หลังงานศพ
- **Starting Status**: ชิองจมอยู่กับความเศร้า
- **Ending Status**: ผู้ร่วมทดลองกลับมาทำลายเทป — พูดแทงชิอง: "ถ้าเธอสำคัญขนาดนั้น แกควรเปิดประตูให้ได้เองสิ" — ชิองฆ่าโดยไม่ได้ตั้งใจ — ซ่อนศพ
- **Key Theme**: จุดที่เขาข้ามเส้น

### Extra E04: กลุ่มสอง
- **POV**: ชิอง มาโดกะ
- **Timeline**: 2 ปีก่อน
- **Starting Status**: ชิองลังเลที่จะรับผู้ทดสอบใหม่ — ยังมีโอกาสหยุด
- **Ending Status**: ผู้หญิงคนหนึ่งร้อง "เปิดประตูให้ฉันที" เสียงเหมือนอายาเมะ — ชิองคลั่ง — ฆ่าด้วยเจตนาครั้งแรก
- **Key Theme**: ครั้งแรกที่เขาฆ่าโดยตั้งใจ

### Extra E05: บ้านที่เรียนรู้คำตอบ
- **POV**: ชิอง มาโดกะ (อายุ 26 → 27)
- **Timeline**: 1 ปีก่อน — เชื่อมโยงสู่ปัจจุบัน
- **Starting Status**: กลุ่มสาม (4 คน) ได้ยินข่าวลือขึ้นไปบ้าน — ตายทั้งหมด — บ้านตอนนี้เป็นระบบฆ่าที่สมบูรณ์แบบ
- **Ending Status — Emotional Bridge สู่เรื่องหลัก**: หลังกลุ่มสามตาย ชิองนั่งอยู่ในห้องควบคุม ท่ามกลางร่องรอยของเหยื่อที่เขาสะสมมาสามปี — ไม่มีความรู้สึกของชัยชนะ มีแต่ความว่างเปล่า
  จากนั้นตัดภาพมาที่เรียวกังวันหนึ่ง ชิองเห็นรายชื่อแขกใหม่ในสมุด:
  > อาราอิ เรียวยะ
  > อามามิยะ นาโอะ
  > ซากุราบะ ไคโตะ
  > อาซาฮินะ ริสะ
  > คุโรซาวะ มิซากิ
  
  เขาอ่านรายชื่อซ้ำ — ห้าคน วัยมหาลัย มาเที่ยวพักร้อน
  ในหัวเขาได้ยินเสียงอายาเมะจากเทป:
  > "กลับบ้านกันเถอะ ชิอง"
  
  เขาปิดสมุด
  แล้วเปิดอีกครั้ง
  แล้วจำชื่อทุกคนได้ขึ้นใจ
  
  เขาไม่ได้กลับ
  เขาเลือกที่จะอยู่ต่อ
  และนี่คือครั้งสุดท้ายที่เขามีโอกาสเลือก
- **Key Connection**: ศพกลุ่มสาม + ร่องรอย = สิ่งที่กลุ่มพระเอกพบในบทที่ 7 — และการที่ชิองจำชื่อทุกคนได้ คือเหตุผลที่เขารู้จักกลุ่มตั้งแต่ก่อนพวกเขาจะรู้จักเขา
